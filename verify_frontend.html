If you're still stuck

  1. Make sure the backend server has been restarted (so it picks up the cache-control fix on /legacy).
  2. Open the browser's DevTools (F12) → Network tab → check "Disable cache" while DevTools is open.
  3. Hard refresh again with Ctrl + Shift + R.
  4. Try opening the tool in an Incognito/Private window to avoid cached state.
